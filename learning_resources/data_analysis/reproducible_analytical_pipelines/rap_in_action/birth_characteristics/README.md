# Analysis for Action: Birth characteristics Reproducible Analytical Pipeline (RAP) example

This repository showcases an early iteration of converting legacy statistical code into a Reproducible Analytical Pipeline (RAP). It was originally used to support the Birth Characteristics and Births by Parents’ Characteristics publications.

Although this version is no longer in use (the work has since been integrated into a larger package), it remains a valuable example of how to modernise existing codebases using RAP principles.

This repository has been reviewed and edited with input from the original RAP development team. It’s intended as a **learning resource** and should be used in conjunction with the **Analysis for Action: Birth Characteristics RAP learning resource**.

> Please note: This codebase is not functional and is provided for demonstration purposes only.

## Repository structure
The repository contains the following key components:
- `old_code/`: Legacy code used in previous publications, included for reference. Referred to as "Original code" in the Analysis for Action learning resource.
- `RAP/`: The main RAP structure, including configuration files and the primary analysis script. Referred to as "RAP" in the Analysis for Action learning resource.
    - `birth.char/`: An R package developed to encapsulate functions and data processing steps for the Birth Characteristics analysis.
